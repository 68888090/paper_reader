# CLIP: Contrastive Language-Image Pretraining

## 创作时间与刊物
- **时间**: 2020年11月19日 (arXiv:2011.10566v1)
- **刊物**: arXiv preprint
- **作者**: Alec Radford, Jong Wook Kim, Christopher Hallacy, Aditya Ramesh, Gabriel Goh, Sandhini Agarwal, Girish Sastry, Amanda Askell, Pamela Mishkin, Jack Clark, Gretchen Krueger, Ilya Sutskever
- **单位**: OpenAI

## 面向的问题与方法

### 研究问题背景
在计算机视觉领域，传统的图像分类模型（如 ImageNet 训练的模型）存在以下局限性：

1. **类别固定性**: 模型只能识别训练时定义的类别，无法零样本泛化到新的类别
2. **类别'{限制**: 即使使用 Open Images 等大规模数据集，类别数量仍然有限（通常 < 1000 类）
3. **专有性**: 模型只能识别训练数据中明确标注的属性，无法理解更广泛的视觉概念

### 解决方案概述
作者提出了 CLIP (Contrastive Language-Image Pretraining)，一种通过对比学习从自然语言监督中预训练视觉模型的方法：

1. **零样本迁移**: 训练后的模型可以直接用于零样本图像分类，无需任务特定的微调
2. **跨模态对齐**: 学习图像和文本的联合嵌入空间，使得语义相关的图像和文本在嵌入空间中接近
3. **可扩展性**: 通过大规模数据集（WIT）进行预训练，模型可以理解广泛的视觉概念

## 方法流程

### 3.1 数据集创建

#### WIT (WebImageText) 数据集
- **规模**: 4亿对图像-文本配对数据
- **来源**: 从互联网上抓取的图像和伴随文本
- **构建方法**:
  1. 从多个来源收集图像-文本配对数据
  2. 过滤低质量的配对
  3. 去除重复和 Near-duplicate 样本
  4. 使用启发式规则过滤恶意或低质量内容

#### 与现有数据集的对比
- **ImageNet**: 14M 张图像，1000 个类别，人工标注
- **JFT-300M**: 300M 张图像，18K 个类别，弱监督
- **CC-3M**: 3M 张图像， crawled 文本
- **WIT (CLIP)**: 400M 张图像，开放式文本描述

### 3.2 前处理

#### 图像预处理
1. **调整尺寸**: 将图像调整为 224×224 像素
2. **归一化**: 使用 ImageNet 的均值和标准差进行归一化
3. **数据增强**:
   - 随机裁剪
   - 随机水平翻转
   - 颜色抖动（可选）

#### 文本预处理
1. **大写规范化**: 将文本转换为标准格式
2. **字节对编码 (BPE)**: 使用 40,000 个合并操作的 BPE 算法
3. **截断**: 将文本截断为 77 个 token

### 3.3 前向传播

#### 模型架构

**双编码器架构**:
1. **图像编码器 (Image Encoder)**:
   - 使用 Vision Transformer (ViT) 或 ResNet
   - ViT 配置: patch size 14, embedding dim 1024, 16 heads
   - ResNet 配置: 50-layer with attention pooling

2. **文本编码器 (Text Encoder)**:
   - 使用 Transformer 编码器
   - 12 层 Transformer
   - 512 维 embedding，16 个注意力头

#### 对比学习目标

**嵌入空间**:
- 图像编码器输出: \( f_I(x) \in \mathbb{R}^{d} \)
- 文本编码器输出: \( f_T(y) \in \mathbb{R}^{d} \)
- 通过对齐后的表示计算余弦相似度

**对比损失**:

给定 N 个图像-文本配对 \(\{(x_i, y_i)\}_{i=1}^{N}\)，计算相似度矩阵:
- \( S_{i,j} = f_I(x_i)^T f_T(y_j) / \tau \)
- 其中 \(\tau\) 是温度参数

**两个任务**:
1. **Image-to-Text**: 给定图像，预测正确的文本
   \[
   \mathcal{L}_1 = -\frac{1}{N} \sum_{i=1}^{N} \log \frac{\exp(S_{i,i})}{\sum_{j=1}^{N} \exp(S_{i,j})}
   \]

2. **Text-to-Image**: 给定文本，预测正确的图像
   \[
   \mathcal{L}_2 = -\frac{1}{N} \sum_{i=1}^{N} \log \frac{\exp(S_{i,i})}{\sum_{j=1}^{N} \exp(S_{j,i})}
   \]

**总损失**:
\[
\mathcal{L} = \frac{\mathcal{L}_1 + \mathcal{L}_2}{2}
\]

#### 零样本分类
对于零样本分类，给定 C 个类别描述 \(\{c_k\}_{k=1}^{C}\) 和测试图像 \(x\):

1. 计算图像嵌入: \( f_I(x) \)
2. 计算每个类别描述的文本嵌入: \( f_T(c_k) \)
3. 计算类别概率:
   \[
   p(k|x) = \text{softmax}(f_I(x)^T f_T(c_k) / \tau)
   \]

### 3.4 反向传播

#### 优化器
- **Adam 优化器**: β1=0.9, β2=0.999, ε=1e-8
- **学习率调度**: warmup + cosine decay
- **批次大小**: 256K 到 327K（取决于模型大小）
- **训练步数**: 300 万步

#### 分布式训练
- 使用多卡分布式训练
- 使用 NVIDIA Apex 进行混合精度训练
- 使用 ULM 对优化器状态进行分片

#### 正则化
- 权重衰减: 0.1
- Gradient clipping: 1.0
- Dropout: 0.0（Transformer 中未使用）

### 3.5 后处理

#### 零样本推理
1. 对测试图像进行前向传播得到图像嵌入
2. 计算与所有类别文本嵌入的相似度
3. 通过 softmax 得到类别概率分布
4. 输出概率最高的类别

#### 特征提取
- 图像编码器输出的特征可用于下游任务
- 通过线性探针评估特征质量

## 关键贡献

1. **大规模对比学习**: 在 4 亿图像-文本配对上进行对比学习，证明了可扩展性
2. **零样本迁移**: 首次在如此大规模上展示零样本图像分类的可行性
3. **多模态对齐**: 证明了图像和文本可以在同一嵌入空间中对齐
4. **ResNet 和 ViT 的对比**: 在 CLIP 训练中比较了两种架构的性能
5. **开源**: 发布了训练好的模型和代码

## 核心创新点

### 1. 数据规模的突破
- 使用 4 亿对图像-文本数据（比之前的 CC-3M 大 100 倍）
- 证明了在更大规模数据上训练可以持续提升性能

### 2. 对比学习的简单性
- 相比,options 如 ViT 的目标函数更简单
- 不需要复杂的损失函数或架构修改

### 3. 自然语言作为监督信号
- 使用自然语言描述作为监督信号
- 文本描述可以表达更丰富的视觉概念

### 4. 零样本分类的可行性
- 证明了零样本迁移在大规模数据上是可行的
- CLIP 在 ImageNet 上达到 76.2%top-1 准确率（零样本）

### 5. 双编码器架构
- 图像和文本编码器独立工作
- 通过对比学习对齐嵌入空间
- 支持灵活的推理模式（图像检索、文本检索、分类）

## 实验结果

### 零样本 ImageNet 分类
- **ResNet-50**: 65.6% top-1
- **ViT-B/32**: 71.1% top-1
- **ViT-B/16**: 75.2% top-1
- **ViT-L/14**: 78.4% top-1
- **ViT-L/14 @ 336px**: 80.2% top-1
- **Ensemble**: 82.3% top-1

### 下游任务微调
- 在多个数据集上微调后达到 SOTA 水平
- 证明了 CLIP 特征的迁移能力

### 对比其他方法
- CLIP 零样本性能超越了许多 supervised 模型
- 在 JFT-300M 上预训练的模型需要微调才能达到类似性能

## 性能指标

| 模型 | ImageNet 零样本 | ImageNet 微调 |
|------|----------------|---------------|
| ResNet-50 | 65.6% | 82.8% |
| ViT-B/32 | 71.1% | 79.4% |
| ViT-B/16 | 75.2% | 81.3% |
| ViT-L/14 | 78.4% | 83.2% |

## 总结

CLIP 是一篇里程碑式的论文，证明了:
1. 通过对比学习可以从自然语言监督中学习通用视觉特征
2. 零样本迁移在大规模数据上是可行的
3. 图像和文本可以在统一的嵌入空间中对齐

该工作开启了"基础模型"时代，影响了后续的 DALL-E、DINO、MAE 等工作。
