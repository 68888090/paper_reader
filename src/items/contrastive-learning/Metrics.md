## DIA-CLIP (arXiv:2602.01772) 指标汇总

### 核心性能

| 指标 | 变化 |
|------|------|
| 蛋白质鉴定数量 | **+45%** |
| 诱饵鉴定数量（假阳性） | **-12%** |

### 与 DIA 分析工具对比

DIA-CLIP 在多个基准数据集上一致优于现有工具（DIA-NN、Spectronaut、MaxDIA 等），在蛋白质鉴定深度和假阳性控制两个维度同时取得显著提升。

### DIA-CLIP 模型架构

| 组件 | 说明 |
|------|------|
| 肽段编码器 | Transformer 编码氨基酸序列 |
| 谱图编码器 | 编码质谱碎片离子特征 |
| 解码器 | 从谱图嵌入重建肽段序列 |
| 对比损失 | InfoNCE / NT-Xent（肽段↔谱图对齐） |
| 生成损失 | 谱图 → 解码器 → 肽段重建损失 |

### 推理方式

| 方法 | 训练方式 | 推理方式 |
|------|----------|----------|
| DIA-NN / Spectronaut / MaxDIA | 每次实验半监督训练 | PSM 重打分 |
| **DIA-CLIP** | **一次预训练** | **零样本 PSM 推断** |

### 应用场景

| 场景 | 优势 |
|------|------|
| 大规模人群队列蛋白质组学 | 无需逐次训练，直接推理 |
| 单细胞蛋白质组学 | 高灵敏度鉴定 |
| 空间蛋白质组学 | 跨条件泛化 |
| 跨物种研究 | 预训练模型直接迁移 |

### 最佳实践建议

1. **预训练一次，多次使用**: 预训练后无需逐次实验重新训练
2. **跨物种迁移**: 预训练模型可直接应用于新物种数据
3. **假阳性控制**: DIA-CLIP 在提升鉴定深度的同时降低假阳性


## Pixel-level Counterfactual Contrastive Learning (arXiv:2603.17110) 指标汇总

### 分割性能

| 方法 | DSC (肺分割) | 备注 |
|------|-------------|------|
| 直接在银标准标签训练 | 基准（较低） | 基线方法 |
| DVD-CL（无监督） | 优于其他密集 CL | 无标注对比预训练 |
| S-DVD-CL（监督） | **~94%** | 银标准标签引导 |
| S-MVD-CL（监督） | **~94%** | 银标准标签引导 |

### DVD-CL vs MVD-CL

| 方法 | 表示质量 | 分割下游 |
|------|----------|----------|
| DVD-CL | ✅ 学到有意义的像素嵌入 | 有效 |
| MVD-CL | ❌ 未能学到有意义的表示 | 失败 |

### 反事实生成配置

| 参数 | 值 |
|------|-----|
| 生成模型 | HVAE (Hierarchical VAE) |
| 因果父节点 | 性别、扫描仪类型、PE 存在性 |
| 反事实类型 | 变扫描仪、变 PE、同时变两者 |
| 总视图数 | 4（1 原始 + 3 反事实） |
| HVAE 训练 | 对比学习前单独预训练，后续固定 |

### 传统增强叠加

| 增强类型 | 参数 |
|----------|------|
| 随机旋转 | 是 |
| 随机裁剪 | 是 |
| 颜色抖动 | 是 |
| 高斯模糊 | 是 |
| Solarization | 是 |

### 对比学习方法对比维度

| 方法 | 对比粒度 | 数据增强 | 监督信号 | 应用任务 | 最佳指标 |
|------|----------|----------|----------|----------|----------|
| CLIP | 图像级 | 标准增强 | 文本 | 零样本分类 | 76.2% Acc |
| DIA-CLIP | 跨模态 | 肽段-谱图配对 | 跨模态自监督 | 蛋白质组学 | +45% 蛋白鉴定 |
| GroupViT | 区域级 | 标准增强 | 文本 | 语义分割 | 46.3 mIoU |
| **本文** | **像素级** | **反事实+标准** | **银标准标签(可选)** | **医学分割** | **~94% DSC** |

### 训练配置

| 参数 | 值 |
|------|-----|
| 优化器 | AdamW |
| 学习率调度 | 余弦衰减 |
| 梯度裁剪 | 是 |
| 对比损失 | NT-Xent |
| 温度参数 | τ（固定） |
| 预训练阶段 | 无标注数据，像素级对比学习 |
| 微调阶段 | 少量金标准标注，分割解码器 |

### 最佳实践建议

1. **DVD-CL 优先于 MVD-CL**: 双视图逐对对比优于多视图联合对比
2. **反事实生成对分布偏移有效**: 当测试数据存在采集/病理变化时，反事实增强显著提升鲁棒性
3. **银标准标签 + 对比学习 > 直接训练**: 噪声标签通过像素级对比学习框架能发挥更大价值
4. **CHRO-map 用于调试**: 可视化像素嵌入可帮助判断对比学习是否学到有意义的表示
