# 2603.17110: 像素级反事实对比学习用于医学图像分割

## 创作时间与刊物
- **时间**: 2026年3月17日 (arXiv:2603.17110v1)
- **刊物**: arXiv 预印本，投稿于 CVPR/ICCV 等计算机视觉顶会级别
- **作者**: Marceau Lafargue-Hauret, Raghav Mehta, Fabio De Sousa Ribeiro, Mélanie Roschewitz, Ben Glocker（帝国理工学院计算机系）
- **主题**: Contrastive Learning（对比学习） + Medical Image Segmentation（医学图像分割）

## 面向的问题与方法

### 研究问题背景

1. **标注数据稀缺**：图像分割依赖大规模像素级标注数据集，标注成本极高、速度慢。银标准（AI 生成）标签虽容易获取，但存在偏差风险。

2. **传统增强的局限**：当前对比学习中使用的标准数据增强（旋转、裁剪、颜色抖动、模糊等）无法模拟真实世界中复杂的数据分布偏移（如不同扫描仪参数、病理变化等）。

3. **反事实对比学习的空白**：已有工作将反事实生成与对比学习结合用于图像级分类，但尚未扩展到像素级密集预测任务（如分割）。

4. **具体医学场景**：胸腔积液（Pleural Effusion, PE）会使肺部大面积不透明，导致模型对肺部区域分割不足（under-segmentation）。

### 解决方案概述

提出将**反事实生成**（Counterfactual Generation）与**密集对比学习**（Dense Contrastive Learning）相结合的框架。核心思路：
- 使用深度结构因果模型（DSCM）中的 HVAE 生成反事实图像作为数据增强
- 设计双视图（DVD-CL）和多视图（MVD-CL）密集对比学习方法
- 提出监督变体（S-DVD-CL, S-MVD-CL）利用银标准标签
- 提出 CHRO-map 可视化算法用于理解像素级嵌入

## 方法流程

### 数据集创建

- 论文基于已有的胸部 X 光片数据集进行实验，聚焦于肺部分割任务。
- **银标准标签**：使用 AI 生成的肺部分割标注作为银标准标签。
- **金标准标签**：使用人工标注的肺部分割标签用于最终评估。
- 数据涵盖正常和带有胸腔积液（PE）的胸部 X 光片。

### 前处理（数据增强）

数据增强分为两层：

**1. 反事实生成（核心增强）**
- **生成模型**：使用层级变分自编码器（Hierarchical VAE, HVAE），基于深度结构因果模型（DSCM）框架。
- **因果图定义**：在胸部 X 光片的 SCM 中，选择三个因果父节点：
  - 性别（sex）
  - 扫描仪类型（scanner type）
  - 胸腔积液存在性（PE presence）
- **干预生成**：通过干预因果变量生成三类反事实图像：
  - (i) 仅改变扫描仪类型
  - (ii) 仅改变胸腔积液存在性（添加或去除 PE）
  - (iii) 同时改变扫描仪和 PE 存在性
- HVAE 在训练对比学习框架之前单独预训练，生成时保持固定不变。

**2. 传统增强叠加**
在反事实基础上进一步施加标准增强：
- 随机旋转（Random Rotation）
- 随机裁剪（Random Cropping）
- 颜色抖动（Color Jittering）
- 高斯模糊（Gaussian Blurring）
- Solarization

最终形成：1 个原始视图 + 3 个反事实视图 = 4 个增强视图。

### 前向传播

**模型架构**
- **编码器 Backbone**：标准卷积编码器（如 U-Net 编码器部分），用于提取像素级特征嵌入。
- 编码器对每个视图分别前向传播，输出像素级特征图。

**四种密集对比学习方法**

#### (a) DVD-CL（双视图密集对比学习，无监督）
- 以非反事实原始图像作为**锚点视图**（anchor view），反事实图像作为**目标视图**（target views）。
- **正样本对**：锚点视图与目标视图中**相同空间位置**的像素。
- **负样本对**：所有其他不同位置的像素组合。
- 继承自 VADeR 的像素级对比框架，扩展支持多视图。
- 对每个目标视图分别计算 NT-Xent 损失，沿两个方向（锚→目标、目标→锚）取平均，再对所有目标视图取平均。

#### (b) MVD-CL（多视图密集对比学习，无监督）
- 同时考虑所有视图（原始 + 3 个反事实），在所有视图间两两计算像素相似度。
- 配对规则与 DVD-CL 相同（相同空间位置为正，其余为负），但扩展到整个多视图集合。
- 在所有视图间联合优化对比损失。

#### (c) S-DVD-CL（监督双视图密集对比学习）
- 利用银标准分割标注重新定义正负样本对。
- **正样本**：属于同一语义类别的像素（如所有属于左肺的像素互为正样本）。
- **负样本**：属于不同语义类别的像素（如左肺 vs 右肺、肺 vs 背景）。
- 仅以非背景像素作为锚点，确保监督信号有意义。
- 在每对视图间计算对比损失后取平均。

#### (d) S-MVD-CL（监督多视图密集对比学习）
- 在 MVD-CL 基础上引入标注信息，所有视图同时参与对比。
- 配对策略与 S-DVD-CL 相同（基于类别而非空间位置）。
- 在所有视图间联合优化。

### 反向传播

**损失函数**
- **基础损失**：NT-Xent（归一化温度缩放交叉熵），温度参数 τ。
- DVD-CL 损失：对每对视图双向计算 NT-Xent 后取平均。
- MVD-CL 损失：在所有视图对之间联合计算 NT-Xent。
- 监督变体额外利用银标准标签过滤正负样本对。

**梯度流动**
- 反事实图像由固定的预训练 HVAE 生成，不参与梯度反向传播。
- 梯度仅通过编码器主干网络更新。
- 反事实生成器的参数在对比学习阶段完全冻结。

**优化器配置**
- 优化器：AdamW
- 学习率调度：余弦衰减（Cosine Learning Rate Decay）
- 使用梯度裁剪（Gradient Clipping）防止训练不稳定。

### 后处理

**下游分割微调**
1. 使用对比学习预训练的编码器作为主干网络。
2. 在编码器后接入分割解码器（decoder）。
3. 使用少量金标准标注数据进行有监督微调。
4. 输出最终像素级分割掩码。

**评估指标**
- Dice 相似系数（Dice Similarity Coefficient, DSC）作为主要评估指标。

**CHRO-map 可视化**
论文提出的新型像素嵌入可视化算法：
1. 使用 UMAP 将高维像素嵌入降维至 2D。
2. 计算二维嵌入的最小体积包围椭圆。
3. 确定将该椭圆映射到单位圆的仿射变换。
4. 将变换应用于所有嵌入，按极坐标分配 HSV 颜色：
   - 角度 θ → 色调（Hue）
   - 半径 r → 明度（Value）
5. 将颜色叠加回原始图像，生成高分辨率彩色覆盖图。

## 关键贡献

1. **首次将反事实生成引入像素级密集对比学习**：填补了从图像级分类到像素级分割的空白，提出四种方法（DVD-CL, MVD-CL, S-DVD-CL, S-MVD-CL）。

2. **反事实增强提升鲁棒性**：通过模拟扫描仪变化和病理变化（PE）的反事实图像，使模型学习对采集和病理变化不变的特征表示。

3. **银标准标签有效利用**：监督变体在银标准标签辅助下，性能优于直接在银标准标签上训练的模型，证明对比学习框架能更好地利用噪声标签。

4. **CHRO-map 可视化工具**：提出新的像素嵌入可视化方法，直观展示嵌入语义结构。

## 实验结果

### 主要结果

| 方法 | DSC |
|------|-----|
| 直接在银标准标签上训练 | 基准 |
| DVD-CL（无监督） | 优于其他密集对比学习方法 |
| S-DVD-CL / S-MVD-CL（监督） | **~94% DSC** |

### 关键发现

1. **DVD-CL 优于 MVD-CL**：CHRO-map 显示 MVD-CL 未能学到有意义的表示，而 DVD-CL 能根据像素相对脊柱的位置有效编码。
2. **监督变体表现最佳**：S-DVD-CL 和 S-MVD-CL 能清晰区分左右肺，在困难 PE 数据上达到约 94% DSC。
3. **反事实增强有效**：相比仅使用传统增强，反事实增强显著提升了模型对扫描仪和病理变化的鲁棒性。

## 总结

本文提出了像素级反事实密集对比学习框架，核心创新在于：(1) 利用反事实生成模拟真实世界的数据分布偏移（扫描仪、病理变化），(2) 设计双视图和多视图像素级对比学习方法，(3) 通过银标准标签监督进一步提升性能。

**优点**：
- 方法简洁有效，首次打通反事实+密集对比学习的技术路线
- 94% DSC 证明了在医学图像分割中的实用价值
- CHRO-map 提供了像素嵌入的可解释性

**潜在局限**：
- 依赖预训练 HVAE 的反事实生成质量
- 仅在胸部 X 光片肺分割上验证，泛化性待验证
- MVD-CL 失败原因缺乏深入理论分析
