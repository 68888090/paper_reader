# DIA-CLIP: 面向零样本DIA蛋白质组学的通用表示学习框架

## 创作时间与刊物
- **时间**: 2026年2月2日 (arXiv:2602.01772v1)
- **刊物**: arXiv 预印本
- **作者**: Yucheng Liao, Han Wen, Weinan E, Weijie Zhang
- **单位**: 北京大学机器学习研究中心、AI for Science Institute、蛋白质组学国家重点实验室
- **主题**: Contrastive Learning（对比学习）应用于蛋白质组学

## 面向的问题与方法

### 研究问题背景

数据非依赖采集质谱（DIA-MS）是蛋白质组学分析的核心技术。当前 DIA 分析流程存在以下局限：

1. **需逐次半监督训练**: 现有的 PSM（肽段-谱图匹配）重打分方法（如 Percolator、mProphet、XGBoost）需要在每次实验运行中单独训练半监督模型。
2. **过拟合风险**: 半监督训练在单次实验内进行，容易过拟合到该次实验的特定条件。
3. **缺乏跨物种/跨条件的泛化能力**: 模型无法在不同物种、不同实验条件之间迁移。

### 解决方案概述

提出 **DIA-CLIP**，将 DIA 分析从半监督训练范式转变为**通用跨模态表示学习**范式：

1. **双编码器对比学习** + **编码器-解码器架构**：建立肽段与对应谱图特征的统一跨模态表示
2. **零样本 PSM 推断**：预训练后无需逐次训练，直接进行肽段-谱图匹配
3. **跨物种泛化**：预训练模型可应用于不同物种和实验条件

## 方法流程

### 数据集创建

- 基于公开 DIA-MS 蛋白质组学数据集构建训练数据
- 包含多种物种、多种实验条件下的质谱数据
- 训练数据为肽段序列-谱图特征对

### 前处理

- **质谱数据预处理**：保留时间（RT）校准、提取离子色谱图（XIC）生成
- **谱图特征提取**：从 DIA-MS 数据中提取碎片离子信号特征
- **肽段序列编码**：将肽段氨基酸序列转为 token 序列，用于文本编码器
- **谱图编码**：将谱图特征（碎片离子 m/z 和强度）转为固定维度表示

### 前向传播

#### 模型架构

```
肽段序列 → 肽段编码器 → 肽段嵌入
                              ↘
                       对比学习对齐
                              ↗
谱图特征 → 谱图编码器 → 谱图嵌入
```

1. **肽段编码器**：类似文本 Transformer，编码氨基酸序列
2. **谱图编码器**：编码质谱谱图的碎片离子特征
3. **编码器-解码器架构**：从谱图嵌入重建肽段序列，增强跨模态对齐
4. 使用 CLIP 风格的双编码器进行跨模态对比学习

### 反向传播

**损失函数**：
- **对比损失（InfoNCE/NT-Xent）**：对齐肽段嵌入和谱图嵌入，使匹配的肽段-谱图对在嵌入空间中靠近，不匹配的远离
- **解码器重建损失**：谱图嵌入 → 解码器 → 重建肽段序列，确保谱图嵌入保留肽段信息

**优化器**：
- 标准深度学习优化器（Adam/AdamW）
- 大规模预训练后固定模型参数

### 后处理

- **零样本 PSM 推断**：计算肽段嵌入与谱图嵌入的余弦相似度，直接判别匹配
- **FDR 估计**：使用 target-decoy 策略估计假发现率

## 实验结果

### 主要结果

| 指标 | 提升 |
|------|------|
| 蛋白质鉴定数量 | **+45%** |
| 诱饵鉴定数量（entrapment） | **-12%** |

### 关键优势

1. **零样本泛化**：预训练后无需逐次实验训练
2. **跨物种/跨条件迁移**：模型可泛化到未见过的物种和实验条件
3. **高精度**：蛋白质鉴定数量大幅提升，同时假阳性降低
4. **应用广泛**：适用于单细胞蛋白质组学、空间蛋白质组学等前沿场景

## 关键贡献

1. **范式转变**：将 DIA 分析从半监督逐次训练转变为通用预训练-零样本推理
2. **跨模态对比学习**：首次将 CLIP 风格的图像-文本对比学习思想迁移到肽段-谱图跨模态匹配
3. **双编码器 + 编解码器融合**：同时使用对比学习和生成式学习，增强跨模态对齐质量
4. **显著性能提升**：45% 蛋白质鉴定提升 + 12% 假阳性降低

## 总结

DIA-CLIP 是对比学习在生命科学领域的重要应用。它借鉴了 CLIP 的跨模态对比学习思想（图像↔文本 → 肽段↔谱图），将蛋白质组学分析从半监督训练转变为通用表示学习。核心价值在于：一次预训练，跨物种/跨条件零样本应用，大幅提升了蛋白质组学分析的效率和准确性。

**与 CLIP 的关联**：DIA-CLIP 是 CLIP 思想在质谱蛋白质组学中的直接迁移，证明了对比学习作为跨模态对齐通用框架的强大能力。
