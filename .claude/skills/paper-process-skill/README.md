# Paper Process Skill

This skill integrates paper reading, paper selection, and paper analysis into a complete workflow.

## Workflow

1. **Clean src/tmp/** - Clear all files before starting
2. **Process papers one by one**:
   - Download paper to src/tmp/
   - Read and extract paper content
   - Analyze and generate summary
   - Update comparative files
3. **Clean src/tmp/** - Clear all files after completion

## Usage

When user provides multiple paper URLs:
1. Clean src/tmp/
2. Process each paper sequentially
3. Clean src/tmp/

## Key Rules

- Always clean src/tmp/ before starting
- Process papers one at a time (sequential)
- Clean src/tmp/ after completion
- Support both Chinese and English themes
